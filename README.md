Este projeto foi desenvolvido em **Kotlin** como parte de um trabalho escolar e tem como objetivo criar um **sistema completo de gerenciamento para a drogaria +PHARMEI**, totalmente integrado ao **Firebase**.

O aplicativo permite **cadastrar, visualizar, editar e excluir dados**, oferecendo um fluxo completo de **CRUD**, com foco principal no gerenciamento de **funcionários** da drogaria.

##

### Autenticação de Usuários
- Login e cadastro através do **Firebase Authentication**.
- Armazenamento seguro das credenciais.
- Controle de acesso para usuários autorizados.

##

### Gerenciamento de Dados (CRUD Completo)

O sistema utiliza exclusivamente o **Firestore Database** para armazenar os dados dos funcionários da +PHARMEI.

#### Dados armazenados:
- Nome  
- E-mail  
- Idade  

#### Operações disponíveis:
- Cadastro de funcionários  
- Atualização de dados  
- Exclusão de registros  
- Consulta de funcionários cadastrados  

##

### Integração com Firebase

#### Firebase Authentication
Responsável por:
- Login  
- Cadastro  
- Gerenciamento de credenciais  

#### Firestore Database
Utilizado para:
- Armazenar funcionários  
- Editar dados  
- Consultar registros  

#### Realtime Database
- Atualizações em tempo real  
- Notificações internas  

##

#### Tecnologias Utilizadas

- Kotlin  
- Jetpack Compose  
- Firebase Authentication  
- Firestore Database  
- Realtime Database (opcional)  
- Android Studio
